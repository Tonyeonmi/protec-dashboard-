이 대시보드는 어떤 목적으로 만들었나요?
저희 팀은 아직 실제 전환율 데이터를 수집한 경험이 없어, 현재 화면의 전환율 수치는 다수의 마케팅 논문에서 참고한 벤치마크 가정값을 기반으로 대입한 시뮬레이션입니다.

이 제안의 핵심은 수치가 아니라 구조입니다.
고객 세그먼트를 나누는 논리 흐름과 세그먼트별 메시지 템플릿을 한 화면에서 확인하고 수정할 수 있는 플로우를 직접 구현해 보여드리는 것이 목적입니다.

클라이언트 분들이 보유하신 실제 전환율 데이터를 대입하시면 시뮬레이션이 바로 업데이트됩니다.
또한 향후 새로운 변수 기준으로 세그먼트를 재정의하실 수 있도록 변수 선택 기능과 기준 흐름표를 함께 제공해드립니다.

섹션 2. 세그먼트 세부 기준 설명
A그룹 청약자격 기준
A그룹은 이미 청약 의향이 있으므로, 지금 당장 실행 가능한지(자격)가 핵심 변수입니다. 특별공급 / 1순위 / 2순위 / 무순위로 나누면 자격별 최적 전략과 타입 추천이 가능합니다.
B그룹 구매목적 + 분양일정
B그룹은 조건이 맞으면 청약할 의향이 있습니다. "왜 사려 하는가(구매목적)"와 "이미 알고 있는가(분양일정 인지)"의 조합이 설득 온도를 결정합니다. 이미 알고 있는 투자+실거주 고객과 처음 접하는 실거주 고객은 전혀 다른 메시지가 필요합니다.
C그룹 구매목적 기준
C그룹은 현재 청약 의향이 없으므로, 장기적 가치 소구가 핵심입니다. "무엇을 위해 살 가능성이 있나(구매목적)"를 기준으로 실거주/투자/투자+실거주로 나눠 미래 시점에 맞는 메시지 톤을 설계합니다.
